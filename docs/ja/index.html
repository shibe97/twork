〜 テキストベースのシンプルなTwitterクライアント 〜

Download for Macintosh Download for Windows

Motivation

Twitterは情報収集ツールとして非常に優秀です。特に技術系の情報はTwitter上に多く存在しています。 あくまで情報収集として仕事中に使いたいという気持ちがありますが、社内で堂々と閲覧しているのは憚られます。
そこで、シンプルで作業画面の隣においてもあまり目立たないTwitterクライアントを作成しました。 タイムラインにアイコン画像を含まない、ありそうで無かったTwitterクライアントです。

How to Download

このページトップのリンクからダウンロードすることができます。
Windows、Mac共にダウンロード可能です。
※未署名のため、ダウンロード後は「右クリック」→「開く」という手順で開く必要があります。

  • ホームタイムライン
    あなたとあなたのフォローしているユーザーのツイート一覧が表示されます。ストリーム形式で閲覧することができます。
  • メンション
    他のユーザーがあなたに対してメンションしたツイートの一覧が表示されます。
  • リストタイムライン
    あなたのリストに登録されているユーザーのツイートやリツイートの一覧が表示されます。ストリーム形式で閲覧することができます。
  • 検索機能
    ツイッターの検索機能が使えます。キーワードによってはタイムラインの流れが速すぎるため、ストリーム形式には現状対応しておりません。
